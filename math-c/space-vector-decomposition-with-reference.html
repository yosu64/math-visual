空間ベクトルの分解

ベクトル p を、3方向の移動 s a, t b, u c の合成として見るデモです。
p = s a + t b + u c
スライダー
段階表示
薄い矢印と薄い箱は s=t=u=1 の見本です。
濃い矢印が今の値です。
表示
色の意味
■ 赤：s a
■ 緑：t b
■ 青：u c
■ 黒：p
■ 薄色：s=t=u=1 の見本
緑の b は、原点からの基底ベクトルも薄く表示しています。
それを平行移動して L→M に置いたものが、分解で使う緑の矢印です。
見方
O → L が s a
L → M が t b
M → P が u c
O → P が p
薄い図は s=t=u=1 の見本
